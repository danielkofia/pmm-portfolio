# Daniel Kofi Agyei — PMM Portfolio

## Deploy to Vercel (10 minutes, free)

### Option A — Drag and Drop (Easiest)
1. Go to vercel.com and sign up with GitHub or Google
2. Click "Add New Project"
3. Click "Deploy without a Git repository" (or drag the folder)
4. Upload ALL files in this folder
5. Click Deploy
6. Your site is live at yourname.vercel.app

### Option B — Vercel CLI
```bash
npm i -g vercel
cd portfolio-deploy
vercel
```
Follow the prompts. Done.

## Files in this folder
- `index.html` — Main portfolio (rename from okokroko-pmm-portfolio-v2.html)
- `daniel-kofi-agyei-cv.pdf` — CV (linked from portfolio)
- `og-image.png` — Social sharing preview image (1200x630)
- `vercel.json` — Vercel configuration
- `README.md` — This file

## After deploying
1. Copy your Vercel URL (e.g. danielkofiagyei.vercel.app)
2. Update the portfolio link in index.html — search for "[Add Vercel link when live]" and replace
3. Update the CV PDF the same way
4. Redeploy

## Custom domain (optional)
In Vercel dashboard → Settings → Domains → Add your domain
